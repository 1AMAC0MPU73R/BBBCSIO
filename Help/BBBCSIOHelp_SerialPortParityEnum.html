The SerialPortParityEnum Enum

The SerialPortParityEnum enum defines the possible parity bit for the bytes transmitted via the serial ports on the Beaglebone Black

 

Source Code

The source code for this class is available online for download and also in browseable form.

 

Syntax

    public enum SerialPortParityEnum
    {
        PARITY_NONE,
        PARITY_ODD,
        PARITY_EVEN
    }
