The InterruptEventHandlerMM Delegate

The InterruptEventHandlerMM delegate defines the data contract which is used to send event information to the objects which subscribe to an InterruptPort's OnInterrupt event.

 

Source Code

The source code for this class is available online for download and also in browseable form.

 

Syntax

public delegate void InterruptEventHandlerMM(GpioEnum evGpio, bool evState, DateTime evDateTime, EventData evData);

 

Example Code

Example code which demonstrates the use of this delegate can be found on the Interrupt Port Example page.
